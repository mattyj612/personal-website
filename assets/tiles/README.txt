HOMEPAGE TILE IMAGES
====================

Drop one square image per page in this folder, named after the page's slug:

    university-courses.jpg
    walking-tours.jpg
    interact.jpg
    history-of-science.jpg
    ... etc.

The slug for each page is defined in /js/site.js.

- Any common web image format works (.jpg, .png, .webp). Update the file name
  in site.js if you use something other than .jpg (via the `tile` field), or
  just keep the default and use .jpg.
- Square images look best (they're cropped to a square automatically).
- Until you add a real image, the site shows a neutral placeholder with the
  page's initials, so the layout always looks intentional.
